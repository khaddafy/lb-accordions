Kintassa Accordions - A Wordpress Plugin

Introduction

This plugin provides an easy, intuitive interface for creating accordions -- stacked panels with titles, which reveal the chosen panel when clicked or otherwise interacted with. Features include:

  • Multiple Accordions
  • Multiple Panels per accordion
  • Horizontal and vertical accordions
  • Add any wordpress content to the accordion panels
  • Even nest accordions inside other accordions!
  • Plugin architecture for new/alternative rendering methods

Shortcodes and tags

Render a Accordion in post/page content using the kintassa_accordion shortcode, as follows:

[kintassa_accordion id="1"]

This will render the accordion from the admin that has the given id, according to the settings chosen in the admin area.

In php code, use the kintassa_accordion tag/function, as follows:

<?php echo kintassa_accordion(1); >
